Medjat

Written knowledge, structured and transmitted. Medjat is the public interface for collections of the Iroko Historical Society. It presents records across datasets while making stewardship boundaries visible rather than silent.

Enter the Ewé Collection Search Collections

mdꜣt / medjat: written knowledge, document, record. In this project, the term names a governed layer of public access built from the ontology, not a neutral storage site.

What Medjat does

Library-wide search framework

Use this as the future shared search view. In the current phase, the live search experience remains inside the Ewé collection.

Stewardship model. Medjat is designed so that public users can encounter both knowledge and its boundaries. Restricted material is not silently erased. Where appropriate, the interface signals that a governed field exists and names the conditions under which it is stewarded.
Architecture. The ontology defines structure. Medjat expresses records. Future authentication layers can mediate differentiated access for community, initiated, and practitioner views without changing the conceptual model.
